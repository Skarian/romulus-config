# romulus-config

## Getting Started

Option 1: Use the provided config file in the Romulus app via this URL:

- `https://raw.githubusercontent.com/Skarian/romulus-config/main/source.json`

Option 2: Create your own config from scratch and optionally use the simulator

- Start with [build-your-own-config.md](./docs/build-your-own-config.md)
- Learn about the simulator here: [simulator.md](./docs/simulator.md)

## Layout

- `source.json`: maintainer provided Romulus config
- `simulator/`: local browser-based simulator
- `docs/`: getting-started, reference, and simulator workflow docs
- `references/romulus/`: pinned upstream Android app reference
